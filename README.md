En este proyecto 2018 se acumulan pruebas e ideas para NURO 2020.

## Tecnologías utilizadas.

### Bulma.

Pasos.
1- Customizar css, añadiendo colores, fuentes propias del proyecto NURO.

Via Sass CLI :
Con la línea de comandos, utilizando Sass.

```

```
